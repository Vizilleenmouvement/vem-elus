📊 Base de données (SQLite)

👥 Élus
26 conseillers municipaux. Nom, rôle, photo, email, commission.
📁 Projets
91 projets du programme. Statut, avancement, thème, jalons, contacts, documents, presse, journal.
📅 Agenda
Réunions : bureau, commissions, conseil. Date, heure, lieu, type.
📝 Comptes rendus
CR de réunions avec participants, ODJ, contenu, fichiers joints.
📚 Bibliothèque
Documents classés par dossiers thématiques et types. Public/privé.
💬 Chat
Messagerie par canaux (Général, Bureau, commissions). Archivage.
📰 Articles
Veille documentaire. Titre, URL, source, résumé, tags.
🔴 Signalements
Incidents terrain. Type, urgence, lieu, historique de statut.
🔐 Connexions
Journal d'accès. IP, date, succès/échec. Rétention 90 jours.

🔐 Sécurité

Authentification
• Cookie session HttpOnly (durée 7 jours)
• Identifiant : prenom.nom
• Support Basic Auth pour les appels API
• Déconnexion auto après 30 min d'inactivité
Protection
• Anti-brute-force : 5 tentatives / 15 min par IP
• Journalisation de toutes les connexions
• Token de réinitialisation temporaire (1h)
• Upload : noms de fichiers assainis
